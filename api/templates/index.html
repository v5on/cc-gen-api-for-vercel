API Documentation

            

                

                    
GET /gen

                    
Description: Generates test credit card numbers based on the provided BIN, with optional expiry date and CVV.

                    
Use Case: Create valid test credit card numbers for development and testing purposes, supporting both AMEX and non-AMEX cards.

                    
Parameters:

                    

                        
                            

                                Parameter
                                Type
                                Description
                                Default
                            

                        
                        
                            

                                bin
                                String
                                Card BIN (6-15 digits or up to 16 with 'x')
                                Required
                            

                            

                                month
                                String
                                Expiry month (MM, e.g., 06)
                                Random
                            

                            

                                year
                                String
                                Expiry year (YYYY or YY, e.g., 2027)
                                Random
                            

                            

                                cvv
                                String
                                CVV (3 or 4 digits)
                                Random
                            

                            

                                amount
                                Integer
                                Number of cards to generate (1-2000)
                                10
                            

                        
                    

                    
Example Requests:

                    
GET /generate?bin=515462&limit=2&month=12&year=28&cvv=000

                    
GET /generate?bin=515462&limit=2&month=12&year=28

                    
GET /generate?bin=515462&limit=2

Response Fields:

                    
    
                        
  • status: Indicates success or error
    • 
                        
  • bin: Input BIN
    • 
                        
  • amount: Number of cards generated
    • 
                        
  • cards: Array of generated cards (card_number|month|year|cvv)
    • 
                        
  • Bank: Issuing bank name
    • 
                        
  • Country: Issuing country with flag
    • 
                        
  • BIN Info: Card scheme and type
    • 
                        
  • api_owner: Developer handle
    • 
                        
  • api_updates: Updates channel
    • 
                    

                    
Notes:

                    
    
                        
  • BIN can include 'x' for random digits or full numbers.
    • 
                        
  • AMEX cards have 15 digits and 4-digit CVV; others have 16 digits and 3-digit CVV.
    • 
                        
  • Expiry date can be provided as MM/YYYY or MM/YY.
    • 
                        
  • Invalid inputs return a 400 error with a descriptive message.
    • 
                        
  • Contact @bro_bin_lagbe for API access and usage details.
